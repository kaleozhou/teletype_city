### 🚀 如何开始游戏

1. **启动服务器**（如果还没启动）：
   ```bash
   source venv/bin/activate
   python3 start_server.py
   ```

2. **连接游戏**：
   ```bash
   # 使用Python测试客户端
   python3 test_telnet.py
   
   # 或者使用真实的telnet客户端
   telnet localhost 2323
   ```
